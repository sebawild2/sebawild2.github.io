Unit 9: Suchbäume

This unit covers fundamental dynamic list data structures:

  • binary heaps
  • binary search trees
  • 2-3 trees
  • red-black trees

Further sources

This unit mostly follows

  • Sedgewick & Wayne, Algorithms, 4th Edition

There are many websites with animations of data structures; a particularly nice for classic data structures is

  • Kováč et al. Gnarley Trees

However, there seems nothing around using precisely the implementation of left-leaning red-black trees (or 2-3 trees) from

  • Sedgewick, Left-Leaning Red-Black Trees, 2008

hence the new one linked above.
